Informativa sulla Privacy

Introduzione

Questo è un sito web personale statico, ospitato sulla piattaforma GitHub Pages e gestito da Sonia Guerci (titolare dello studio di consulenza informatica Eurisoft con sede a Tavagnacco, Udine).


Gestione del Consenso e Cookie

Questo sito adotta un sistema di gestione del consenso preventivo (tramite Google Tag). Ciò significa che nessun cookie analitico o di profilazione di terze parti viene caricato o memorizzato sul tuo dispositivo fino a quando non esprimerai un consenso esplicito e positivo tramite il banner presente all'accesso.


Statistiche di Navigazione (Google Analytics)

Previo tuo consenso esplicito, utilizziamo Google Analytics (GA4) per analizzare in modo anonimo e aggregato l'utilizzo del nostro sito statico. GA4 è impostato per anonimizzare l'indirizzo IP dei visitatori prima di qualsiasi memorizzazione dei dati sui server di Google Ireland Limited.

Puoi revocare il consenso in qualunque momento reimpostando le preferenze del tuo browser o scaricando il componente aggiuntivo del browser per la disattivazione di Google Analytics.


Hosting (GitHub Pages)

Il sito è ospitato sull'infrastruttura di GitHub Inc. Durante la navigazione, la piattaforma potrebbe registrare dati tecnici standard (log di sistema come indirizzi IP temporanei) esclusivamente per finalità di sicurezza informatica e per garantire l'erogazione del servizio di hosting. Per maggiori informazioni, puoi fare riferimento alla GitHub Privacy Statement.
